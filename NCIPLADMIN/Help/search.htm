Search

The Admin Tool homepage is the Search page.

Search options can be combined. For example, the keyword field can be combined with criteria in one or more of the fields below.

Available search options

•Filter by Live Interfaces (interfaces on which a publication is active; default is to search all pubs)
•Keyword search on free text fields
•NIH #
•Multiple select fields including Cancer Type, Subject, Audience, Language, Publication Format, Collections, Race, Book Status, Reading Level, ROO Most Common List, Awards
•Filter by ROO Most Common List

Examples

Search for a specific publication by number

Type P123 in the keyword (Search) field.

The publication record displays in View mode.

Search for a type of publication

Select Cancer Type: Breast and Subjects: Clinical Trials and Coping with Cancer.

(To select two Subject values, hold the CTRL key on the keyboard while clicking each value in the field.)

Several rows of search results display, showing publications relevant to breast cancer, clinical trials, and coping.

Filter a search

Select Cancer Type: Breast and Subjects: Clinical Trials and Coping with Cancer.

Then, select the check box for Live Interfaces: Exhibit.

The search results displayed are limited to publications that are active in the Exhibit ordering interface.
