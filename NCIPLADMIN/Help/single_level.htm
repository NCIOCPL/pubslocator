Single Level Lookups

Each single level lookup page contains a section to add a new lookup value and a section to view and edit existing values.

Add a new lookup item

1.Type a new lookup item description and any related information for the lookup value.
•For example, to add a new Language, type a Language Description.
2.Click the Add button to add the lookup item to the system.
3.The new lookup item displays in the Existing section below. The Enabled state of the new lookup item shows as Active.

Edit an existing lookup item

1.Click the Edit link of the lookup item you want to edit. Update and Cancel links will display.
2.Change the item information.
3.Click Update to update the information, or click Cancel to cancel the action.

Change whether an existing lookup item is active or inactive

Make an item inactive

If the existing item is Active, click the Inactive link.

A message confirms that you want to inactivate the item. Click OK.

The existing item is now displayed as Inactive.

Make an item active

If the existing item is Inactive, click the Active link.

A message confirms that you want to activate the item. Click OK.

The existing item is now Active.
