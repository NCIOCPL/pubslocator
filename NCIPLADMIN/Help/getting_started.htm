Getting started

Getting to the Admin Tool

The Admin Tool is part of CISnet. A separate login is not needed.

Production URL: https://cissecure.nci.nih.gov/extranet/ncipubsadmin/

User permissions

There are 3 levels of user permission:

•Administrator
•Regular User
•Data Warehouse Staff

Many parts of the Admin Tool are available only to Administrators. Regular User and Data Warehouse Staff users do not have access to add or edit most parts of publication records.

Navigating the Admin Tool

The Admin Tool has a top navigation menu which can be used to navigate sections including Record, Search, Reports, Lookups, Help, and Logout.

The Record, Reports, and Lookups sections have further drop-down options.

Some areas are not available to Regular User and Data Warehouse Staff level users.

Logging out of Admin Tool

Click the Logout link in the top navigation menu to close the window and return to CISnet.
