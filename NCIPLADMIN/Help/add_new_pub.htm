Add/Edit Publication

Publication records are managed through the Add/Edit Publication page.

Sections of a publication record

•Core Summary including CPJ and Prod History
•Core Details (expandable/collapsible section)
•Interface and additional tabs

Core Summary

The Core Summary section at the top of the page has some fields that are read-only and some that are editable.

Read only fields

•NIH #
•Quantity Available
•Book Status
•Date Last Pub Received
•Original Publication Date
•Date Last Printed
•Date Last Publication Revised
•Date Archived
•Original Pub Date
•Max Qty/Title
•Weight

Editable fields

•Pub ID
•Short Title
•Long Title
•FS #
•Spanish Title Accents
•Spanish Title No Accents
•URL
•Contents & Covers URL

Core Details

The Core Details is an expandable/collapsible section. The section contains fields and list boxes used to select multiple values associated with the Product or materials.

Fields in this section

•Description
•Copyrighted Material
•Keywords
•Thumbnail File Name
•Summary
•# of Total Pages
•Dimension
•Color
•Other
•POS Instruction
•Cancer Type
•Audience
•Language
•Publication Format
•Collections
•Race/Ethnic Group
•Reading Level
•Awards

Interface section/additional tabs

When editing each tab in this section, click the Save or Cancel button before moving to a different tab.

Tabs are available in this section to control options specific to the interfaces NCIPL, ROO, Exhibits, and Catalog.

Adding receivings

The Prod History section can be used to add information about received publications after the publication record has been created.

Saving historical information

The attachments and comments tabs can be used to save information such as e-mails and copies of the Add Inventory form.

Related products

The related products tab can be used to associate this publication with other products.

Currently, any products that are a Related Product for this publication will display on the Publication Details page in NCIPL.

Translations

The Translations tab can be used to associate this publication with a version in another language.

Live interfaces

The live interfaces checkboxes control the ordering interfaces in which the publication record can be live (available).

A publication record will only be made live in an ordering interface if the live interfaces checkbox is selected AND it meets all criteria for display. For example, the publication record will not be made live in NCIPL if only the display status "online" is selected for NCIPL but no URL is entered.
