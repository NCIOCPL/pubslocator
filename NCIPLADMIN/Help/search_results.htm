Search Results

The Search Results page displays the results based on the criteria selected in the previous search page.

The results can be sorted by clicking column titles. Displayed results can be refined and exported to an Excel spreadsheet. Individual records can be viewed by clicking the linked publication title.

To view a record

Click the linked publication title to open an individual record in View mode.

To sort results

Click a column title to sort by that value (Publication ID, Publication Title, Qty Available, Book Status, or Date Created)

To select or remove results

•Select products from the result table with individual checkboxes or the Select All checkbox next to the Date Created column
•Click the Keep Selection(s) button to retain only the selected results
•Or, click the Drop Selection(s) button to remove the selected results

To save results as a spreadsheet

Click the Excel icon at the top or bottom right of the search results table to export the results to a spreadsheet.

To search again

•Click Refine Search to go back to the search page with the previous search criteria
•Click New Search to go back to the search page and start over
