Lookups (Administration)

Each lookups section can be used to view and modify active and inactive codes.

Types of lookups

The lookups can be divided into 2 groups of lookup tables that are managed in a similar way. The first group, single level lookups, contains the following lookup tables: Audience, Awards, Cancer Type, Language, Product Format, Reading Level, Race, Series, and Subject. The second group, nested level lookups, contains the following lookup tables: Subcategory and Sub subcategory.

Spell checking

Whenever there is a free text description field for adding or editing a lookup value, a spell check menu will appear at the top and bottom of the page.

To spell check an entry, select English or Spanish from the language drop-down menu, then click the "abc" button.
