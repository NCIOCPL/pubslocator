Using this Help file

Using the Contents

The Contents (to the left of this page) lists all the pages in Help.

•To view a topic, click the topic name.
•To expand a closed topic, click the closed book icon next to the topic name.

Using the Index

The Index (to the left of this page) lists the keywords for the pages in Help.

•To view the Index, click the Index link next to Contents.
•Click a letter of the alphabet to skip to that letter in the Index.
•Click a keyword to view associated topics.
•If only one page matches the keyword, the page displays in the panel on the right. If more than one page matches the keyword, a list of matching pages displays. Click the page you want to view, and it displays in the panel on the right.
