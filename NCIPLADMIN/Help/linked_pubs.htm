Linked Pubs

Linked Pubs are linked though one Primary Publication and one or more Secondary Publications. The linked Pubs are identified by the ID of the Primary Pub.

Linked Pubs are a one-directional relationship and are based on a specific ordering interface or interfaces. When the Primary publication is ordered through this interface, the Secondary publication or publications are included by the system. However, if a Secondary publication is ordered separately, this does not affect the Primary publication. An example of a Linked Pub might be a Primary publication that has an Errata sheet as a secondary publication.

Create a new Linked Pub

1.Enter a Primary linked pub ID
2.Select one or more interfaces for the linked pub
3.Click Create New to add the new Primary Pub and go to the Update Linked Pub page
4.Add Pub ID and Quantity for each Secondary Publication that you want to link to the Primary pub, then click Save to add each Secondary Publication.

The list of linked publications displays in a table with the ID, title, and quantity of each pub.

Edit an existing Linked Pub

1.From the Linked Publication page, click the Edit button for a Primary Pub.
2.Add, edit or remove Secondary publications and click Save.

The list of linked publications displays in a table with the ID, title, and quantity of each pub.

•You can click Add New to go back to the main Linked Publication page and add a new Linked Pub or view all Linked Pubs.
