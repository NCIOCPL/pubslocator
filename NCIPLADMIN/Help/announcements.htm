Announcements

From the Announcement page, click the Add New Announcement or Edit Existing Announcement tabs to manage announcements.

Add a new announcement

1.Enter Announcement Name and URL
2.Enter Start and End dates in MM/DD/YYYY date format
3.Click Add button to save the announcement

Announcements are currently published only on the front page of NCIPL.

Edit an existing announcement

1.Click the Edit link of the Announcement you want to edit. Update and Cancel links will display
2.Change Announcement Name, Announcement URL, Start Date, or End Date
3.Click Update to update the information, or click Cancel to cancel the action

Delete an existing announcement

Click the Delete link of the Announcement you want to remove.

A confirmation message displays to verify that you want to remove the announcement.
