Introduction

The Admin Tool homepage is the Search page. From here you can

•Search for a publication to view, edit, or add receivings

Or use the top navigation menu to

•Add a new publication
•Manage groups of records including linked pubs and virtual kits
•Manage data such as announcements, exhibit dates, or the catalog sequence
•Generate reports
•Manage the lookups (the values by which records are categorized)
