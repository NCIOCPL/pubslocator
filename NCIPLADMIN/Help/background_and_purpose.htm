Background and purpose

Inventory maintenance

The Admin Tool is the application used to maintain the inventory. It provides a means of updating publication records that can be displayed in various ordering interfaces.

CPJ

Admin Tool currently works in conjunction with CPJ. Some information about publication records is maintained only in CPJ and is "view-only" in the Admin Tool. Data is synchronized nightly from CPJ to Admin.

Parts of Admin

The Admin Tool currently includes

•an interface for searching publications and displaying results
•screens for adding and editing records
•management of kits (groups of publications)
•management of lookup values
•reports to allow printing and viewing of data
