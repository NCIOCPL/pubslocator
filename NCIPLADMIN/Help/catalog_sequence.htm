Catalog Sequence

Using Catalog Sequence

The Catalog Sequence page has a drop-down menu to choose sections of the catalog.

The first menu level includes Category, Subcategory, and Sub subcategory.

If Subcategory or Sub subcategory is selected in the first menu level, a second drop-down menu level appears to facilitate choosing a type of catalog sequence to rearrange.

Changing the on-page display

By default, the items on the page are shown sorted by Sequence number. You can click the linked column headings to sort by a particular column.

Saving or reverting changes

Click the Reset button to undo changes to a section of the catalog sequence. Click the Save button to save changes.
