Virtual Kits

A Virtual Kit is a group of publications with a "pseudo" product name to identify it as a kit. For example, “Web123” could be the name given to a virtual kit made up of pubs P123, P124, and P125. The kit would appear as a single entity during search in the Admin Tool and would count each individual item for cost recovery according to cost recovery rules per title.

Virtual Kits are a one-directional relationship between the "pseudo" kit ID and the kit publications, and are based on a specific ordering interface or interfaces. When the Kit ID is ordered through this interface, the kit publications are included by the system. However, if a kit publication is ordered separately, it has no impact on the other publications.

Create a new Virtual Kit

1.Enter a new virtual kit "pseudo" name. This must be an ID that is not already in use for a Virtual Kit. A message will prompt you to choose a different name if you choose a name that is already in use.
2.Select one or more interfaces for the Virtual Kit.
3.Click Create New to add the new Virtual Kit and go to the Create Virtual Kit page.
4.Add Pub ID and Quantity for each publication that you want to add to the Virtual Kit, then press Save to add each publication.

The list of publications in the Virtual Kit displays in a table with the ID, title, and quantity of each pub.

•You can click Add New to go back to the main Virtual Kit page and add a new Virtual Kit or view all Virtual Kits.

Edit an existing Virtual Kit

1.From the Virtual Kit page, click the Edit button for a Virtual Kit.
2.Add, edit or remove publications and click Save.

The list of publications in the Virtual Kit displays in a table with the ID, title, and quantity of each pub.

•You can click Add New to go back to the main Virtual Kit page and add a new Virtual Kit or view all Virtual Kits.
