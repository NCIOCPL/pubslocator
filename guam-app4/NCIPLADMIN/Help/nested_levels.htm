Subcategory look up management

Select a category first to bring up a list of related subcategories.

This section contains tabs to add new subcategories and view existing subcategories.

Create a new subcategory

1.Click the Add New Subcategory tab.
2.Type a new lookup item description and any related information for the lookup value.
3.Choose whether to allow Sub subcategories.
4.Click the Save button to add the lookup item to the system.
5.The new lookup item displays in the Existing section.

Edit an existing subcategory

1.Click the Edit Existing Subcategory tab.
2.Pick a Subcategory from the drop-down list. Another set of controls will display.
3.Edit the subcategory information.
4.Click the Save button to save the edits, Cancel to discard modifications, or Delete to delete the Subcategory.
•A confirmation message will display if you choose to Delete the Subcategory.
•You can click the Subcategories link at the top to go back to choose a different Subcategory. This works the same as clicking the Cancel button.
5.If the Subcategory has Sub subcategories, you can click the Edit SubSubCat button to edit its Sub subcategories.
6.Your edits are saved after clicking Save or navigating to Sub subcategories.

Sub subcategory look up management

Create a new Sub subcategory

1.Click the Add New Sub subcategory tab.
2.Type a new lookup item description and any related information for the lookup value.
3.Choose which Subcategory to assign the Sub subcategory to.
4.Click the Save button to add the lookup item to the system.
5.The new lookup item displays in the Existing section.

Edit an existing Sub subcategory

1.Click the Edit Existing Sub subcategory tab.
2.Pick a Sub subcategory from the drop-down list. Another set of controls will display.
3.Edit the Sub subcategory information.
4.Click the Save button to save the edits, Cancel to discard modifications, or Delete to delete the Sub subcategory.
•A confirmation message will display if you choose to Delete the Sub subcategory.
•You can click the Sub subcategories link at the top to go back to choose a different Sub subcategory. This works the same as clicking the Cancel button.
5.Your edits are saved after clicking Save.
