Keywords Report

The keyword report lists all the keywords in use in one or more product records.

It can be used to compare against the existing report received from Austin each month.

To search for a specific record

Enter a publication ID and click the Go button.

You can enter more than one publication ID, using commas to separate the IDs.

To view all keywords

Click the All button to view keywords for all publications.

To export to an Excel spreadsheet

To save this report as a spreadsheet, click the Excel icon.
